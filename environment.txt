## Примечание по шрифтам
# xhtml2pdf встраивает системный шрифт в PDF. Для корректного отображения
# кириллицы скрипт ищет один из распространённых шрифтов:
#   macOS:    /System/Library/Fonts/Supplemental/Arial.ttf
#   Linux:    /usr/share/fonts/truetype/dejavu/DejaVuSans.ttf
#             /usr/share/fonts/truetype/liberation/LiberationSans-Regular.ttf
#   Windows:  C:/Windows/Fonts/arial.ttf
# Если ни один не найден, PDF будет сгенерирован, но кириллица может
# отображаться некорректно.
