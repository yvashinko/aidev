# Программа курса

## Модуль 1. Принципы работы генеративного ИИ (3 ч)
`Gen1/01-llm-fundamentals/`
- Понятие LLM
- Токены и контекст
- Принципы обучения моделей
- Причины возникновения «галлюцинаций»
- Области применения AI в разработке
- Анализ ответов моделей
- Разбор типовых ошибок моделей

## Модуль 2. Prompt Engineering для разработчиков (4 ч)
`Gen1/02-prompt-engineering/`
- Структура эффективного промпта
- Роль system prompt
- Few-shot prompting
- Chain-of-thought подход
- Управление контекстом
- Генерация функции
- Объяснение сложных алгоритмов
- Генерация SQL-запросов

## Модуль 3. Генерация кода и MVP проектов (5 ч)
`Gen1/03-codegen-mvp/`
- Генерация сервисов
- Генерация API
- Генерация CLI-инструментов
- Генерация дизайна
- Генерация MVP
- Создание CLI-инструмента
- Разработка MVP backend

## Модуль 4. Debugging и анализ логов (4 ч)
`Gen1/04-debugging-logs/`
- Анализ stack trace
- Анализ логов
- Поиск root cause
- Генерация исправлений (fixes)
- Анализ состояния системы
- Анализ ошибок production-среды
- Анализ логов сервиса

## Модуль 5. Тестирование и обеспечение качества кода (4 ч)
`Gen1/05-testing-quality/`
- Генерация unit-тестов
- Генерация integration-тестов
- Улучшение покрытия кода
- Property-based тестирование
- Генерация тестов на C++/C#
- Генерация edge-case сценариев

## Модуль 6. Документация и архитектурные решения (3.5 ч)
`Gen1/06-docs-architecture/`
- Генерация README
- Генерация API-документации
- Генерация ADR (Architecture Decision Records)
- Архитектурные обсуждения
- Написание ADR
- Подготовка README
- Описание архитектуры сервиса

## Модуль 7. Локальные модели для разработки (3 ч)
`Gen1/07-local-models/`
- Обзор локальных LLM
- Работа с Ollama
- Работа с LM Studio
- Модели для генерации кода
- Запуск локальной модели
- Проведение code review локально
- Data governance, security, TCO

## Модуль 8. RAG: Retrieval-Augmented Generation (3.5 ч)
`Gen1/08-rag/`
- RAG для работы с документацией и кодовой базой
- Поиск по документации
- Поиск по репозиториям
- Подключение внутренней базы знаний
- Поиск по кодовой базе
- MCP (Model Context Protocol)
- Оценка качества retrieval
