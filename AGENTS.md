# AGENTS.md — правила работы с репозиторием курса

## Стек и формат

- Markdown, pandoc, Make, GitHub Actions.
- Каждая глава — `Gen1/<модуль>/<NN>-chapter.md`.
- Конспект занятия — `<NN>-notes.md` в той же папке.
- Шаблоны — `templates/`, промпты — `prompts/`, пример — `examples/`.

## Стиль

- Декларативный тон, формальное «вы», нет маркетинговой лексики.
- Числа и единицы вместо качественных оценок.
- Trade-offs вместо односторонних рекомендаций.
- `_[as of YYYY]_` для быстро устаревающих фактов.

## Definition of Done для новой главы

- [ ] Глава следует шаблону из `.cursor/skills/course-author/SKILL.md`.
- [ ] В каждом содержательном разделе есть `TL;DR`, `See also` и практическое следствие.
- [ ] В конце главы есть контрольные вопросы, Quick reference, глоссарий, доп. материалы.
- [ ] `make lint` и `make check-links` не находят новых ошибок.
- [ ] PDF пересобран (`make pdf`) после значимых изменений.

## Don'ts

- Не добавлять сгенерированные PDF в исходники — только в `dist/`.
- Не дублировать общие правила внедрения между файлами; ссылаться на `Gen1/teaching-guide.md`.
- Не оставлять шаблоны с `do_stuff()` или псевдо-псевдокодом.
